# homebrew-tap

Homebrew tap for [**lulu-lumei-dock**](https://github.com/vinlee19/lulu-lumei-dock) —
a macOS menu-bar "Dynamic Island" for local AI coding agents.

```bash
brew tap vinlee19/tap
brew install --cask lulu-lumei-dock
```

The cask strips the quarantine attribute after install (the app is ad-hoc signed, not
notarized), so it opens without a Gatekeeper prompt.
